## Мониторинг и безопасность

- **Мониторинг:**
  - Grafana — визуализация метрик (CPU, RAM, трафик).
  - Sentry — отслеживание ошибок в реальном времени.

- **Безопасность:**
  - Секреты хранятся в GitHub Secrets или Vault.
  - Доступ к staging через SSH и VPN.
  - Проверка зависимостей через Snyk.
